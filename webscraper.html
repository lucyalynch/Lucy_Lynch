Team Project: Web Scraper

Github page

This project was a collaberation between Jonas Blanchard, Jonah , and myself. It was an assignment for the IB Computer Science class, and was created to scrape the Amazon best-sellers page.

Tools

Twilio
BeautifulSoup

Techniques

Variables
Conditionals
Create/read files

Twilio and BeautifulSoup were the main tools used in the development of the project.
Twilio was chosen as it is allowed the program to send notifications to the user’s phone.

BeautifulSoup allowed easier access to the elements of the page’s HTML code, allowing us to easily find the price and URL of the top toy on Amazon.

Variables are a core part of programming, and were used throughout the app to store URLs, authentication tokens, file names, and more.

If and else statements were also used inside the app, which provided the some of the logic of the algorithm.

Finally, files were created and read, which allowed it to store information between launches.
